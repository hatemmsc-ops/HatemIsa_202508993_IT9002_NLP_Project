# HatemIsa_202508993_IT9002_NLP_Project
Hatem Isa's NLP sentiment analysis assignment for IT9002 at Bahrain Polytechnic (ID: 202508993). The project uses text preprocessing, Bag of Words, N-grams, and TF-IDF to analyze Amazon product reviews. Multinomial Naive Bayes and Logistic Regression are used for sentiment classification and assessment.
